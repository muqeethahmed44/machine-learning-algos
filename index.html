Supervised Learning: Learns from labeled data
Unsupervised Learning: Finds patterns in unlabeled data
Linear Regression
Supervised
Tabular Data
Predicts continuous numerical values by finding the best-fit line through data points. The algorithm learns the relationship between input features and target values to make predictions on new data.
Example Applications:
• House price prediction based on size and location
• Sales forecasting from historical data
• Temperature prediction from weather features
Decision Tree
Supervised
Tabular Data
Creates a tree-like model of decisions based on features. Each internal node represents a test on a feature, branches represent outcomes, and leaf nodes represent final predictions or classifications.
Example Applications:
• Customer churn prediction in telecommunications
• Loan approval decision systems
• Medical diagnosis based on symptoms
Random Forest
Supervised
Tabular Data
Combines multiple decision trees to improve accuracy and reduce overfitting. Each tree votes on the final prediction, and the majority vote determines the outcome. Works by training each tree on random subsets of data and features.
Example Applications:
• Credit card fraud detection
• Customer behavior prediction in e-commerce
• Stock market trend analysis
Support Vector Machine
Supervised
Tabular Data Computer Vision
Finds the optimal boundary (hyperplane) that separates different classes with the maximum margin. Works well for both linear and non-linear classification by using kernel functions to transform data into higher dimensions.
Example Applications:
• Email spam classification
• Handwritten digit recognition
• Cancer detection from medical imaging
Convolutional Neural Network (CNN)
Supervised
Computer Vision
Specialized neural network designed for processing grid-like data such as images. Uses convolutional layers to automatically learn spatial hierarchies of features, from simple edges to complex objects.
Example Applications:
• Facial recognition systems
• Autonomous vehicle object detection
• Medical image analysis for disease diagnosis
Recurrent Neural Network (RNN)
Supervised
NLP
Neural network architecture designed for sequential data. Maintains memory of previous inputs through feedback loops, making it ideal for time-series and text data where order matters.
Example Applications:
• Language translation services
• Speech recognition systems
• Stock price prediction using historical data
Transformer
Supervised
NLP Generative AI
Uses self-attention mechanisms to process entire sequences simultaneously rather than sequentially. Captures long-range dependencies in data and forms the foundation of modern language models like GPT and BERT.
Example Applications:
• ChatGPT and other conversational AI systems
• Document summarization and translation
• Code generation and completion
Generative Adversarial Network (GAN)
Supervised
Computer Vision Generative AI
Consists of two neural networks competing against each other: a generator creates synthetic data while a discriminator evaluates authenticity. Through this adversarial process, the generator learns to create increasingly realistic outputs.
Example Applications:
• Realistic image generation and photo enhancement
• Creating synthetic training data for AI models
• Video game asset generation
K-Means Clustering
Unsupervised
Tabular Data Computer Vision
Partitions data into K distinct groups based on similarity. Iteratively assigns data points to the nearest cluster center and updates centers until convergence. No labels required as it discovers natural groupings.
Example Applications:
• Customer segmentation for targeted marketing
• Image compression by color clustering
• Anomaly detection in network traffic
Principal Component Analysis (PCA)
Unsupervised
Tabular Data Computer Vision
Reduces data dimensionality by identifying principal components that capture the most variance. Transforms data into a new coordinate system where the greatest variance lies along the first coordinates.
Example Applications:
• Data visualization and exploration
• Feature extraction before model training
• Noise reduction in sensor data

Algorithm Relationships and Connections

Supervised vs. Unsupervised

Supervised algorithms require labeled training data and learn to map inputs to known outputs. Unsupervised algorithms find hidden patterns in unlabeled data without predetermined categories.

Tree-Based Methods

Decision Trees and Random Forests share a common foundation. Random Forest improves upon single Decision Trees by combining multiple trees to reduce overfitting and increase accuracy through ensemble learning.

Neural Network Family

CNNs, RNNs, Transformers, and GANs all belong to the neural network family but are specialized for different data types. CNNs excel at spatial data, RNNs handle sequences, Transformers process text efficiently, and GANs generate new data.

Dimensionality Reduction

PCA is often used as a preprocessing step before applying other algorithms. It reduces computational complexity and can improve performance of supervised methods like SVM or Random Forest by eliminating noise.

Domain Specialization

Tabular data typically uses traditional ML algorithms (Linear Regression, Decision Trees, Random Forest). Computer Vision relies on CNNs and GANs. NLP primarily uses RNNs and Transformers. Generative AI combines multiple architectures.

Evolution of Complexity

Algorithms have evolved from simple linear models to complex neural networks. Modern approaches like Transformers build upon earlier innovations in neural networks, incorporating attention mechanisms that revolutionized NLP tasks.
